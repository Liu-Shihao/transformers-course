# Qucik Start

```shell
pip install langchain
```

设置OPENAI_API_KEY
```shell
#langchain默认使用OpenAI Model，设置OPENAI_API_KEY
export OPENAI_API_KEY = "..."
```
或者
```python
import os 
os .environ[ "OPENAI_API_KEY" ] = "..."
```
安装好 LangChain 并设置好环境后，我们就可以开始构建我们的语言模型应用程序了
